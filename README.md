# medical-qna-client

## 목적
의료 질의응답 시스템에 대한 JavaFX client Application
<br> (의료 질의응답 시스템, 운영 대시보드용 서버와 상호작용)

<br>

---


### 주요 기능
| |간단 설명|
| - | - |
|(1)|운영자의 resource & log 모니터링|
|(2)|일반 사용자의 의료 질의응답 서비스|
|(3)|본인의 증상과 유사한 질의응답 요청|

<br>

### 사용하는 기술
| |종류|
|-|-|
|(1)|JavaFX|
|(2)|WebSocket client|
|(3)|Java 비동기 병렬 프로그래밍|
